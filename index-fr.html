Dynamic lstopo

Durant mon stage de Master 1, j'ai réalisé une mise à jour de l'outil hwloc pour afficher des données d'analyse de performance sur l'interface graphique d'origine. Elle utilise les compteurs de performance disponibles dans nos CPU et affiche des données de cache, la charge CPU et le nombre d'instructions exécutées par cycle pour chaque coeur logique. Ce travail est sous licence BSD, tu es donc libre de l'utiliser. Pour plus d'information, regarde le README du repo Bitbucket.

Bomberman en JavaScript

Sur mon temps libre, je travaille sur une version web du vieux jeu Bomberman. Je la voulais mutlijoueur c'est pourquoi j'utilise un serveur en JavaScript à l'aide de Node.js. Pour le moteur de jeu, j'ai décidé d'utiliser Quintus.js. C'est un impressionnant moteur de jeu HTML5 fait en JavaScript. Vous trouverez plus d'information sur ce projet sur son dépôt Github.
